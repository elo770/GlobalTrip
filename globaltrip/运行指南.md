# 运行指南

### 第一步：安装依赖（如果还没安装）

**安装前端依赖：**
```powershell
cd globaltrip
npm install
```

**安装后端依赖：**
```powershell
cd server
npm install
cd ..
```
### 第二步：启动项目*：

#### 终端 1  启动前端服务器

```powershell
cd globaltrip
npm run dev
```

前端将在 **http://localhost:3000** 运行

## 使用说明

1. **加载预设路线**：
   - 在地图规划页面点击"加载改革开放路线"按钮
   - 将自动加载：深圳 → 东莞 → 广州 → 上海 → 北京

2. **查看日程**：
   - 点击顶部菜单"日程视图"
   - 查看按时间线排列的每日行程

3. **管理预算**：
   - 点击顶部菜单"预算管理"
   - 添加预算项，查看统计图表
