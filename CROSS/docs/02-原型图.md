# GlobalTrip 页面原型图（低保真）

## 1. 信息架构
- 首页 `/home`
- 地图规划 `/map`
- 日程视图 `/schedule`
- 预算管理 `/budget`

## 2. 全局导航原型
- 顶部导航：Logo + 首页 / 地图规划 / 日程视图 / 预算管理

## 3. 首页原型
- 欢迎区：标题、副标题、创建新路线、查看示例路线
- 我的路线：路线卡片列表（名称、目的地数、天数、删除）
- 示例路线卡片：改革开放主题 + 加载按钮

## 4. 地图规划页原型
- 顶部工具条：POI 搜索、添加点、平移、放大、缩小、适应范围
- 左侧路线面板：加载示例、清空路线、保存路线
- 目的地列表：排序、删除、缩略信息
- 地图主区域：点位与连线展示
- 路线摘要：目的地数、总距离、预计天数

## 5. 日程视图页原型
- 时间线结构：按天展示城市、活动、住宿、备注
- 顶部按钮：加载改革开放路线

## 6. 预算管理页原型
- 左侧：预算明细表（天数、类型、描述、金额、删除）
- 右侧：总预算、按类型统计、按天统计、图表
- 弹窗：新增预算项（天数、类型、描述、金额）
